### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/iprime2/ecommerce-krishil
```

### Setup .env file

```
REACT_APP_API_TOKEN = 
REACT_APP_API_URL = https://ecommerce-krishil.onrender.com/api
REACT_APP_UPLOAD_URL = http://localhost:1337
REACT_APP_STRIPE_KEY=
```

### Install packages

```shell
yarn 
```

### Start the app

```shell
yarn run start
```

## Available commands
